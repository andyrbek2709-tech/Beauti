Telegram интеграция

1) Создайте бота в BotFather, 2) вставьте token и ваш telegram user id, 3) сохраните и используйте webhook данные ниже.

Как создать бота (быстро)

  1. Откройте Telegram и найдите @BotFather.
  2. Нажмите /newbot и задайте имя бота.
  3. Задайте username (должен заканчиваться на bot).
  4. Скопируйте выданный Bot Token и вставьте в поле ниже.
  5. Узнайте свой Telegram ID (через бота вида @userinfobot) и вставьте в поле ниже.
  6. Нажмите Подключить бота автоматически.
